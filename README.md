# 小审 3.0 - 审计作业平台

> "前线生产车间 + 后台标准展厅" —— 为一线审计人员提供宽松、灵活、数据打通的作业环境

## 项目概述

小审 3.0 是新一代审计作业平台的前端原型设计，采用双视图架构（文件视图 + 业务视图），参考 [PostHog](https://posthog.com/) 的设计理念，结合 Apple 审美标准构建。

## 设计理念

### 双视图架构
- **文件视图** 📁：类似桌面操作体验，适合一线审计人员日常作业
- **业务视图** 💼：传统网页工作台，适合项目经理和管理层查看进度

### 文件视图三大区域
1. **业务管理区**（顶部/侧边导航）：项目导航、搜索、用户操作
2. **文件作业区**（主内容区域）：文件夹管理、底稿编辑、协作
3. **智能助理区**（右侧 IM 面板）：项目组群聊 + AI 数字助理（置顶）+ 工作地图

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 灵活工作区构建 | 支持批量建项、自由命名文件夹、模板挂载 |
| 人员角色与权限 | 继承式授权、动态人员管理 |
| 账套数据关联 | 统一取数、嵌入式查账 |
| Office/WPS协同 | 本地映射编辑、跨表链接支持 |
| IM 即时通讯 | 项目组群聊、私聊、AI 助手置顶推送 |
| AI智能助手 | 交互式产品说明书、情境感知、主动推送预警 |
| 工单驱动协作 | 服务目录、资料完整性校验、成果自动回填 |
| 送审与归档 | 送审映射（中转岛）、组合送审、流程流转 |

## 技术栈

- **HTML5** + **CSS3** + **Vanilla JavaScript**
- 纯前端实现，无需安装任何依赖
- 支持现代浏览器（Chrome / Edge / Safari / Firefox）

## 如何使用

1. 直接用浏览器打开 `index.html` 即可预览
2. 点击右上角 **文件视图/业务视图** 按钮切换两种布局
3. 右侧 AI 面板可以进行模拟对话交互
